プライバシーポリシー / Privacy Policy

最終更新日 / Last updated: 2025年8月6日

1. 取得する情報 / Collected Data

本アプリは、以下の情報を取得・収集しません。
This app does not collect the following:

  • 氏名、住所、メールアドレス、電話番号などの個人情報 / Personal information
  • 位置情報 / Location data
  • 外部サーバーへの通信情報 / Server communications
  • 利用履歴やトラッキングデータ / Usage analytics or tracking

2. アプリ内データの取り扱い / Local Data Handling

登録されたリマインダー情報は端末内にのみ保存され、外部送信は行いません。
All reminder data is stored locally on the user’s device and never transmitted externally.

通知設定はUNUserNotificationCenterを使用し、ユーザーの許可がある場合にのみ動作します。
Notifications are handled via UNUserNotificationCenter and require user permission.

3. 第三者提供 / Third-party Sharing

本アプリは、ユーザーの情報を第三者に提供することはありません。
This app does not share any user data with third parties.

4. データの管理 / Data Management

ユーザーデータはすべて端末に保存され、アプリをアンインストールすると削除されます。
All data is stored on the device and deleted upon uninstallation.

6. 改定について / Changes to this Policy

必要に応じて本ポリシーを変更する場合があります。アプリのアップデートで通知します。
This policy may be updated. Any changes will be communicated through app updates.
